# Enozom Task

## INSTALLATION
<pre>
- git clone https://github.com/Hager-Abd-El-Galil/Enozom-Task
- Composer install 
- cp .env-example .env
- php artisan migrate
- php artisan db:seed
- php artisan serve
</pre>

## TECHNOLOGIES
- Laravel Framework
- MYSQL
